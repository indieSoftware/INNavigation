The following extensions are available globally.

  • Double

    These values are best guesses and might change any time with an iOS update. They are used as a workaround for the lack of a callback after a transition animation has finished. When these values are not correct anymore then transitions might break where screens get blank or not updated. This can be validated with the example app when executing multiple transitions one after each other.
