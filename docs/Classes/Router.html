  • popAfter(index:) Asynchronous

    Removes all views from the current horizontal path after the given index.

    Does nothing if the index is higher than the amount of routes in the list or when negative.

    Will be animated only if exactly one view has been popped.

    Declaration

    Swift

    public nonisolated func popAfter(index: Int) async

    Parameters

    index

    The index of the route from which all following routes will be removed. The index is 0 based with 0 pointing to the root view and thus 1 represents the first pushed screen. Using an index of 0 does the same as popToRoot.
